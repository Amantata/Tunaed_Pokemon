## 종족치 랭크 견본

| 랭크 | 범위 | ++ 기준 |
|------|------|---------|
| S | 200~ | |
| AAA | 170~195(-~+) | AAA++ … 199 |
| AA | 140~165(-~+) | AA++ … 169 |
| A | 110~135(-~+) | A++ … 139 |
| B | 90~105(-~+) | B++ … 109 |
| C | 60~85(-~+) | C++ … 89 |
| D | 40~55(-~+) | D++ … 59 |
| E | 1~35(-~+) | E++ … 39 |

> 예：A+ = 135, A = 111~134, A- = 110
