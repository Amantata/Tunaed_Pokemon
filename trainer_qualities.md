# 【자질의 효과】

트레이너의 자질(지시·통솔·육성·능력) 랭크별 효과를 정의합니다.

---

## ㆍ「지시」

상대보다 「지시」가 1랭크 높을수록 「C+1」.

| 랭크 | 효과 |
|------|------|
| 「D」 | 『4식 6령』. 『지령』은 효과 없음. (트리거로서의 의미만 유효) |
| 「C」 | 『4식 6령』. 『지령』은 기본. |
| 「B」 | 『4식 6령』. 『지령』은 『숙달』 |
| 「A」 | 『4식 8령』. 「타임」 1회. 『지령』은 『숙달』 또는 『전개』. 『저격 지령』『회피 지령』 추가. |
| 「AA」 | 『5식 8령』. 「타임」 2회. 『지령』은 『숙달』『전개』 혼합 가능. 『맞춰라！』 추가. |
| 「AAA」 | 『5식 8령』. 「타임」 3회. 『지령』은 『숙달』『전개』 혼합 가능. |
| 「S」 | 『5식 8령』. 「타임」 4회. 『지령』은 『숙달』『전개』 혼합 가능. |

---

## ㆍ「통솔」

「A」랭크 이상일 때, 『추가 역할』을 임명 가능하다.

| 랭크 | 효과 |
|------|------|
| 「D」 | 기본역할만 사용가능 (강화판역할 사용불가). 『유대』 1개 (일부 예외 존재.) |
| 「C」 | 기본 역할의 강화판을 임명할 수 있다. 『부수』포텐셜 사용가능. 『유대』 1개 |
| 「B」 | 『유대』 1개, 종족치 총합 3100 |
| 「A」(상승 효과에 의해) | 『유대』 2개. (일부 예외 존재.) |
| 「A」(트레이너 순수 자질) | 배틀 전 『역할』『부수』『익스펜션』 변경 가능. 『PT ①』 부여. 『유대』 2개. (일부 예외 존재.) 종족치 총합 3300 |
| 「AA」 | 배틀 전 『역할』『부수』『익스펜션』 변경 가능. 『PT ①』『PT ②』 부여. 「명중 +1」. 『유대』 3개. (일부 예외 존재.) 종족치 총합 3500 |
| 「AAA」 | 배틀 전 『역할』『부수』『익스펜션』 변경 가능. 『PT ①』『PT ②』 부여. 「명중 +2」. 『유대』 4개. (일부 예외 존재.) |
| 「S」 | 배틀 전 『역할』『부수』『익스펜션』 변경 가능. 『PT ①』『PT ②』 부여. 「명중 +3」. 『엠퍼러』 임명 가능. 『왕의 유대』 전원에게 부여 가능. |

> **참고**
> - 『역할』 임명：일부 역할은 포켓몬의 자질이 필요하다. 『체이서』『스카우터』 등.
> - 『익스펜션』 변경：자기의 타입 중에서 선택해서 변경 가능. 특성으로 추가되는 타입은 『익스펜션』을 부여할 수 없다.
>
> ※ 『포텐셜』등으로 증가한 통솔의 경우, 유대 갯수의 추가, 명중+ 효과만 얻을 수 있다.

## ㆍ「육성」

레벨의 차이. 육성 C를 기준으로 1랭크당 ±5. 「±」에 따라 레벨의 격차 등 발생.

| 랭크 | 효과 |
|------|------|
| 「D」 | 최대 『제2계제』, 『천부의 재능』, 『에이스』한정 『제3계제』, PT내 레벨 상승 +10 1체, +5 1체로 변경. |
| 「C」 | 최대 『제2계제』. 『천부의 재능』, 『에이스』의 4계제 불가능 |
| 「B」 | 최대 『제2계제』. 『천부의 재능』, 『에이스』한정 『제4계제』 |
| 「A」 | 최대 『제3계제』. |
| 「AA」 | 최대 『제3계제』. 『PT ①』의 부여 가능. |
| 「AAA」 | 최대 『제3계제』. 『PT ①』의 부여 가능, PT내 2체까지 『PT ②』부여 가능. 『극』화. 「기술머신」을 소지품으로 할 수 있음. |
| 「S」 | 최대 『제3계제』. 『PT ①』『PT ②』의 부여 가능, PT내 3체까지 『PT ②』부여 가능. 『극』화. 「기술머신」을 소지품으로 할 수 있음. |

> **참고**
> - 「기술머신」：해당 기술을 1번 내보낼 수 있는 소비형 소지품. 자신의 PT 멤버(대기 포함)의 기술 중 선택 가능.
> - 동일한 기술의 기술머신은 【소지품 중복】에 해당된다.

---

## ㆍ「능력」

고유/전용 포텐셜의 개수 및 강약. 「±」에 의해 차이 발생.

| 랭크 | 효과 |
|------|------|
| 「D」 | 고유 포텐셜 1개, 전용 포텐셜 1개. |
| 「C」 | 고유 포텐셜 1개, 전용 포텐셜 3개. |
| 「B」 | 고유 포텐셜 1∼2개, 전용 포텐셜 3∼5개. |
| 「A」 | 고유 포텐셜 2∼3개, 전용 포텐셜 6개 이상 (8). |
| 「AA」 | 고유 포텐셜 2∼3개, 전용 포텐셜 6개 이상. (8+) |
| 「AAA」 | 고유 포텐셜 2∼3개, 전용 포텐셜 6개 이상. (8+) |
| 「S」 | 고유 포텐셜 2∼4개, 전용 포텐셜 6개 이상. (8+) |
