## 1. 포켓몬 데이터 구조

포켓몬 데이터는 6개 항목으로 구성됩니다.

| 번호 | 항목 | 설명 |
|------|------|------|
| ① | 이름 | 포켓몬 이름 (별명 포함 가능) |
| ② | 타입 | 최대 2가지 타입 |
| ③ | 특성 | 고유한 패시브 효과 |
| ④ | 소지품 | 들려준 아이템 |
| ⑤ | 기술목록 | 전투에 사용할 수 있는 기술 목록 |
| ⑥ | 능력치 | 6종 스탯의 종족치 랭크 (E/D/C/B/A/AA/AAA/S, +/- 포함) |

## 2. 타입 시스템

### 2.1 18가지 타입

노말, 불꽃, 물, 전기, 풀, 얼음, 격투, 독, 땅, 비행, 에스퍼, 벌레, 바위, 고스트, 드래곤, 악, 강철, 페어리

### 2.2 STAB (자속보정)

같은 타입의 기술을 사용할 경우 위력 ×1.5

### 2.3 타입 면역 — 상태이상

| 타입 | 면역 대상 |
|------|-----------|
| 불꽃 | 화상 |
| 얼음 | 얼음 (상태이상) |
| 전기 | 마비 |
| 풀 | 씨뿌리기 |
| 독, 강철 | 독·맹독 |
| 고스트 | 통상교대 불가 |

### 2.4 타입 상성표 (18×18)

풀→물→불꽃→풀, 격투→바위→비행→격투 등 기본 상성 관계. 게임 원작의 18타입 상성표 적용.

## 3. 특성 시스템

각 포켓몬 고유의 패시브 능력. 배틀 중 자동 발동.

### 주요 특성 예시

| 특성명 | 효과 |
|--------|------|
| 가희 | 소리 기술 위력 ×1.5 |
| 불꽃몸 | 접촉 기술 받으면 30% 확률로 화상 |
| 잔비 | 필드에 나오면 비 날씨 발생 |
| 매직미러 | 상태변화 기술을 반사 |
| 우격다짐 | 추가효과 없음 대신 위력 ×1.3 |
| 초식 | 풀 기술 흡수 → 공격+1 |
| 가속 | 매 턴 종료 시 속도+1 |
| 프레셔 | 필드 진입 시 상대 속도-1 |
| 하늘의은총 | 기술 추가효과 발생률 ×2 |
| 부유 | 공중에 떠있어 땅 기술 무효 |
| 에어록, 날씨부정 | 날씨 효과 무효화 |
| 매직가드 | 직접 공격 이외 피해 무효 |
| 끝의대지 | 큰가뭄 발생 |
| 델타스트림 | 난기류 발생 |
| 미스트메이커 | 필드 진입 시 미스트필드 전개 |
| 지구력 | 공격받을 때마다 방어+1 |

## 4. 기술 시스템

### 4.1 기술 데이터 형식

```
기술명… 분류/타입/위력/명중/대상/우선도/접촉여부/효과
```

### 4.2 분류

| 분류 | 데미지 계산 방식 |
|------|----------------|
| 물리 | 공격 vs 방어 |
| 특수 | 특공 vs 특방 |
| 변화 | 데미지 없음, 효과만 |

### 4.3 대상 종류

자신, 단일, 상대전체, 자신이외, 전체필드, 아군필드, 상대필드

### 4.4 우선도

| 우선도 | 기술 예시 |
|--------|-----------|
| +4 | 방어, 판별, 킹실드, 니들가드 |
| +3 | 속이다 |
| +2 | 신속 |
| +1 | 아쿠아제트 |
| ±0 | 일반 기술 |
| -7 | 한판뒤집기 |

### 4.5 접촉 기술의 특이사항

- 접촉 기술은 상대의 특성(불꽃몸 등)이나 소지품(울퉁불퉁멧 등)을 트리거할 수 있음
- 방호패드 소지 시 접촉 트리거 무효

### 4.6 이중 타입 기술

일부 기술은 2가지 타입을 가짐 (예: 플라잉프레스 = 격투/비행). 더 유리한 상성으로 계산하며 양쪽 효과 모두 적용.

### 4.7 일격필살 기술

- 명중 시 상대 남은 HP만큼 고정 데미지
- 명중률 30 고정; 명중/회피 상승 무효 (단, 노가드/마음의눈 상태는 예외)
- 사용자 레벨 ≥ 상대 레벨 이상이어야 효과 발동
- 기합의띠/버티기로 버티기 가능; 타입 0배 시 무효
- 타입 약화 열매 발동 불가 (고정 데미지이므로)

## 5. 능력치 시스템

### 5.1 6대 능력치

HP, 공격, 방어, 특공, 특방, 속도

### 5.2 종족치 랭크

| 랭크 | 수치 범위 |
|------|-----------|
| S | 200+ |
| AAA | 170~195 |
| AA | 140~165 |
| A | 111~135 |
| B | 90~105 |
| C | 60~85 |
| D | 40~55 |
| E | 1~35 |

- 각 랭크에 +/- 보정 존재 (A+ = 135, A = 111~134, A- = 110)
- HP는 데이터 시트에 미표시 (특별 취급)
- 개체치(IV) 존재, 노력치(EV)는 기본 0

## 6. 능력변화 (랭크 시스템)

### 6.1 변화 가능 능력치 7종

공격, 방어, 특공, 특방, 속도, 명중, 회피

### 6.2 전투 능력치 변화 배율

| 랭크 변화 | 배율 |
|-----------|------|
| +n | (2+n)/2× |
| -n | 2/(2+n)× |

### 6.3 명중/회피 변화 배율

| 랭크 변화 | 배율 |
|-----------|------|
| +n | (3+n)/3× |
| -n | 3/(3+n)× |

- 교대 시 전부 초기화

### 6.4 강화 vs 상승 구분 (ST-02)

- **강화**: 직접 배율 적용 (×2 등), 랭크 단계 무관
- **상승**: 랭크 단계 변화 (+1, +2 등)
- 데미지 공식에서 별도 처리됨

## 7. 급소 (크리티컬)

### 7.1 확률

| 단계 | 확률 |
|------|------|
| 기본 | 1/16 |
| +1 | 1/8 |
| +2 | 1/4 |
| +3 | 1/3 |
| +4 | 1/2 |

### 7.2 효과

- 데미지 ×2 (일부 ×3)
- 공격자의 능력 감소 무시
- 방어자의 능력 상승 무시
- 리플렉터/빛의장막 관통
- 급소는 능력변화로 취급하지 않음

## 8. 상태이상

6종 + 혼란 (실질 7종)

| 상태이상 | 효과 |
|----------|------|
| 독 | 매 턴 종료 시 최대HP 1/8 데미지 |
| 맹독 | 시작 1/16, 매 턴 1/16씩 증가; 교대 시 리셋 |
| 얼음 | 행동 불가; 쾌청/불꽃기술로 해제; 매 행동 1/5 확률 자연해제 |
| 마비 | 속도 ×1/2; 매 행동 1/4 확률로 행동 불가 |
| 잠듦 | 1~4T 행동 불가 (잠자기: 2T); 동시에 양측 2체 이상 잠듦 불가 (2체 잠듦 규칙) |
| 화상 | 매 턴 종료 시 최대HP 1/8 데미지; 물리 공격력 절반 |
| 혼란 | 1~4T; 1/2 확률 자해 (위력 40 물리; 공격 랭크 적용; 리플렉터 무관) |

- 중첩 불가 (동시에 하나만)
- 교대해도 유지 (혼란 제외)

## 9. 상태변화 — 상대에게 적용

### 혼란

- 1~4T 지속; 1/2 확률 자해 (위력 40, 공격 랭크 적용)
- 구애머리띠/테크니션/불가사의부적 영향 없음

### 풀죽음

- 그 턴 남은 행동 스킵
- 멘탈효과

### 저주 (고스트 타입)

- 자신 HP 1/2 감소 → 상대에게 저주 부여 → 상대 매 턴 종료 시 1/4 HP 데미지

### 저주 (비고스트 타입)

- 속도-1, 공격/방어+1

### 씨뿌리기

- 매 턴 종료 시 1/8 HP 흡수당함
- 풀 타입 면역 (이미 풀 타입이 돼도 해제 안 됨)

### 마음의눈

- 상대가 받는 다음 기술이 필중
- 공중날기/구멍파기/다이빙/섀도다이브 상태에서도 적중
- 사용자가 교대하면 효과 사라짐

### 앵콜

- 마지막에 사용한 기술만 사용 가능
- 멘탈효과

### 도발

- 변화기술 사용 불가
- 멘탈효과

### 검은눈빛

- 통상교대 불가
- 사용자가 필드를 나가도 유지
- 고스트 타입 면역
- 유턴/바톤터치/탈출버튼으로는 교대 가능

**멘탈효과 목록**: 헤롱헤롱, 앵콜, 트집, 사슬묶기, 도발, 회복봉인

## 10. 상태변화 — 자신에게 적용 (Part 1)

### 대타출동

- 최대HP 1/4 소비하여 분신 생성; 분신이 대신 데미지 수신
- **차단되는 효과**: 혼란/씨뿌리기/하품/악몽/조이기/마음의눈/금제/회복봉인/특성없음/텔레키네시스/검은눈빛
- **차단 안 되는 효과**: 헤롱헤롱/죽음의카운트/도발/트집/앵콜/사슬묶기/꿰뚫어보기/미라클아이/저주
- **특이**: 길동무/죽음의카운트 → 분신 파괴 + 빈사 동시; 울부짖기/날려버리기는 차단 불가

### 방어 (+4 우선도)

- 해당 턴 대부분의 기술 차단
- **예외**: 방어관통 기술, 저주/변신/미래예지 등

### 길동무

- 자신 상태; 공격받아 쓰러지면 공격자도 빈사
- 기합의띠 무시; 대타출동 차단 안 됨; 상태이상/날씨 데미지에는 발동 안 됨

### 매직코트

- 변화기술을 매직미러처럼 반사

### 공중날기/구멍파기/다이빙/섀도다이브

- 1T 무적 후 공격; 각각 취약한 기술 존재

## 11. 상태변화 — 상대에게 적용 (Part 2)

### 하품 / 동면

- 다음 턴 종료 시 잠듦/얼음 상태
- 대타출동으로 차단 불가 (최종적인 상태이상은 적용됨)
- 2체 잠듦 한계 시 하품 사용은 되나 잠듦은 적용 안 됨

### 회복봉인

- HP 회복 행위 전부 불가 (기술/아이템/상태이상 회복 모두 포함)
- 멘탈효과

### 사슬묶기

- 마지막 사용 기술 사용 불가
- 멘탈효과 (멘탈허브로 해제 가능)
- 구애 상태에서 마지막 기술이 봉인되면 발버둥만 가능

### 봉인

- 특정 기술만 사용 불가; 교대해도 유지 (멘탈효과 아님)

### 떨어뜨리기

- 지면에 내려앉음; 땅 기술 피격; 압정뿌리기 등 즉시 발동

### 텔레키네시스

- 공중에 뜸; 일격필살 제외 모든 기술 필중
- 교대해도 유지 (마음의눈과 다름)

### 송전 / 악인 / 침수

- 그 턴의 자신 기술이 각각 전기/악/물 타입이 됨

### 미래예지 / 파멸의소원

- 다음 턴 종료 시 데미지 적용
- 사용자의 특공으로 계산; 사용자 능력변화 무시
- 방어 관통; 타입 상성 적용
- 필드에 있는 포켓몬이 KO 크레딧 획득

### 폭탄

- 접촉 기술 받으면 상대에게 1/4 HP 데미지; 턴 종료나 피격 후 자동 만료

## 12. 상태변화 — 자신에게 적용 (Part 2)

### 난동부리기

- 2T 동안 같은 기술 강제 사용
- 정상 만료 시 혼란 발동
- 빗나가거나 풀죽음 등으로 중단 시 혼란 없음
- 마이페이스 특성으로 혼란 방지 가능

### 소란피기

- 난동부리기와 유사하나 만료 시 혼란 없음
- 발동 중 필드 전체 잠듦 방지 (방음 특성은 수면 방지 효과 면역)

### 아쿠아링

- 매 턴 종료 시 최대HP +1/8 (큰뿌리로 강화)
- 회복봉인 상태에서 불가

### 충전 / 냉각 / 가열

- 다음 전기/불꽃/얼음 기술 위력 ×2
- 다른 타입 사용하거나 턴 종료 시 소멸

### 전자부유

- 자신이 공중에 뜸; 지면 타입 무효
- 필중 효과 없음
- 떨어뜨리기, 땅 기술 피격, 교대 시 해제

## 13. 상태변화 — 자신에게 적용 (Part 3)

### 작아지기

- 회피+2 효과는 기술 효과
- 상태 자체는 디메리트: 짓밟기/누르기/하드롤러/드래곤다이브/고스트다이브/플라잉프레스가 필중 + 위력×2

### 웅크리기

- 방+1 효과는 기술 효과
- 상태 자체: 구르기/아이스볼 위력 ×2

### 분노

- 공격기술로 데미지 받을 때마다 공격+1
- 연속기 → 맞은 횟수만큼 상승
- 4~5T 지속 (원작과 다름); 부가 데미지는 발동 안 시킴

### 뿌리박기

- 매 턴 1/16 HP 회복; 강제교대 면역
- 통상교대 불가; 공중에 떠있어도 땅 기술 피격
- 전자부유 사용 불가; 바톤터치/유턴/아름다운허물로는 교대 가능
- 교대 후 다음 포켓몬이 압정 등 영향 받는지는 해당 포켓몬 타입 기준

### 비축하기

- 1~3회 중첩; 방/특방 각 +1씩 상승
- 토해내기 위력: 1회=100, 2회=200, 3회=300
- 꿀꺽 회복량: 1/4, 1/2, 전체
- 토해내기/꿀꺽 사용으로 상태 해제 시 방/특방 상승도 사라짐
- 바톤터치로 교대 시 방/특방 상승은 유지, 상태는 인계 안 됨

### 참기

- 2T 동안 행동 불가
- 상태 해제 턴 종료 시, 받은 데미지×2를 마지막 공격자에게 반환
- 물리/특수 무관 반환

### 날따름

- 상대 필드의 단일 대상 기술을 자신에게 유인
- 아군 기술에는 작용 안 함
- 카운터/미러코트/참기의 반환 대상도 자신으로 변경 가능
- 모으기 기술은 실제 공격 턴에만 유인 가능
- 피뢰침/마중물 특성보다 우선
- 자신이 쓰러지면 원래 대상으로 돌아옴

### 파워트릭

- 공격과 방어 실수치를 교환 (능력변화는 유지)

### 조이기

- 통상교대 불가; 매 턴 종료 시 최대HP 1/8 데미지
- 사용자가 필드를 나가도 유지됨

## 15. 필드의 상태 — 자진 효과

아군 필드에 적용되는 효과들. 중첩 불가 (같은 효과 재사용 불가, 시간만 갱신 안 됨).

### 순풍

- 4T 전개 (실질 효과 3T)
- 아군 속도 ×2
- 안개제거로 해제 불가
- 속도 기반 기술 위력도 ×2

### 리플렉터

- 5T; 상대 물리기술 위력 반감
- 급소 시 효과 상실
- 깨트리다 등에는 효과 없음
- 혼란 자해 데미지는 반감 안 됨
- 사이코쇼크형 기술 (특공 vs 방어) 무효
- 안개제거로 해제 가능

### 빛의장막

- 5T; 상대 특수기술 위력 반감
- 리플렉터와 동일한 특이사항

### 오로라베일

- 싸라기눈 날씨 조건 필요
- 5T; 물리+특수 모두 반감

> **안개제거로 해제되는 것**: 리플렉터, 빛의장막, 오로라베일, 신비의부적, 흰안개  
> **안개제거로 해제 안 되는 것**: 순풍, 주술

### 신비의부적

- 5T; 상태이상/혼란/하품→잠듦/동면→얼음 차단
- **예외 (차단 불가)**:
  - 이미 하품 상태 → 잠듦
  - 헤롱헤롱, 봉인, 멸망의노래
  - 난동부리기 종료 → 혼란
  - 특성에 의한 상태이상 (독가시, 불꽃몸 등)
  - 잠자기 (자신이 스스로 사용)
- 안개제거로 해제 가능

### 주술

- 5T; 상대 급소 불가
- 반드시 급소 효과 기술도 차단
- 안개제거로 해제 불가

### 흰안개

- 5T; 상대로부터의 능력 감소 차단
- 자신이 사용한 기술로 인한 감소는 차단 불가
- 안개제거로 해제 가능

### 무지개

- 더블배틀 전용 (불꽃맹세+물맹세)
- 4T; 기술 추가효과 발생률 ×2

## 16. 필드의 상태 — 적진 효과 (설치기술)

### 분류

| 분류 | 특징 |
|------|------|
| **대지설치기술** | 공중에 뜬 포켓몬에 무효; 다른 대지설치기술과 공존 가능 |
| **타입설치기술** | 공중에 뜬 포켓몬도 피격; 다른 타입설치기술과 공존 불가 (덮어씀) |

### 대지설치기술

**압정뿌리기**
- 최대 3회 중첩
- 진입 포켓몬에게 최대HP의 1/8(1회) / 1/6(2회) / 1/4(3회) 데미지
- 공중에 뜬 포켓몬, 검은철구 미소지 예외

**독압정**
- 1회: 독 상태; 2회: 맹독 상태
- 독 타입이 밟으면 흡수하여 해제
- 미스트필드 전개 시 독 적용 안 됨 (미스트메이커 진입 처리 참고)

**끈적끈적네트**
- 진입 시 속도 1단계 감소 (능력변화)

### 타입설치기술

**스텔스록**
- 바위 타입 상성에 따라 최대HP의 (상성)/8 데미지
  - 2배 = 1/4; 4배 = 1/2; 0.5배 = 1/16; 0.25배 = 1/32
- 공중에 뜬 포켓몬도 피격

**습지초원** (더블배틀, 물맹세+풀맹세)
- 4T; 진입 포켓몬 속도 ×1/4

**불바다** (더블배틀, 불꽃맹세+풀맹세)
- 4T; 매 턴 종료 시 불꽃 외 포켓몬 1/8 데미지

## 17. 필드의 상태 — 전체 필드 효과

### 17.1 날씨

날씨 효과는 중첩 불가; 새 날씨가 기존 날씨를 덮어씀.

**기본날씨** (5T, ~바위 소지품으로 8T)

**쾌청 (햇살이 강하다)**
- 불꽃기술 ×1.5; 물기술 ×0.5
- 얼음 상태이상 불가
- 번개/폭풍 명중 50%로 감소
- 솔라빔/솔라블레이드 모으기 없이 사용 가능
- 아침햇살/광합성/달의불빛 회복량 최대HP 2/3
- 성장: +2단계 상승
- 웨더볼 → 불꽃 위력 100

**큰가뭄 (햇살이 아주 강하다)**
- 쾌청 효과 + 물기술 무효
- 사용 포켓몬이 필드에 있는 한 지속; 기본날씨 발동 불가
- 동격 날씨(폭우/난기류)로는 덮어씌울 수 있음

**비**
- 물기술 ×1.5; 불꽃기술 ×0.5
- 번개/폭풍 필중
- 솔라빔 위력 반감
- 아침햇살/광합성/달의불빛 회복량 최대HP 1/4
- 웨더볼 → 물 위력 100

**폭우**
- 비 효과 + 불꽃기술 무효
- 큰가뭄과 동일한 지속/발동 규칙

**싸라기눈**
- 얼음 타입 외: 매 턴 종료 시 최대HP 1/16 데미지
  - 대타출동/공중날기/방어 상태도 피격; 불가사의부적도 방어 불가
  - 방진고글/매직가드/방진/구멍파기/다이빙은 면역
- 눈보라 필중
- 오로라베일 사용 가능
- 아침햇살/광합성/달의불빛 회복량 최대HP 1/4
- 웨더볼 → 얼음 위력 100

**모래바람**
- 바위 타입 특방 ×1.5
- 바위/땅/강철 외: 매 턴 종료 시 최대HP 1/16 데미지 (싸라기눈과 동일 면역 규칙)
- 모래모으기 회복량 최대HP 2/3
- 아침햇살/광합성/달의불빛 회복량 최대HP 1/4
- 웨더볼 → 바위 위력 100

**난기류**
- 비행 타입의 약점(전기/얼음/바위)이 1배로 완화
- 비행 타입 아닌 포켓몬에게는 적용 안 됨
- 거꾸로배틀에서는 역방향으로 적용
- 웨더볼에는 영향 없음
- 동격 날씨로만 덮어씌울 수 있음

### 17.2 룸 계열

**공통 특징**
- 같은 기술을 한 번 더 사용하면 해제 가능
- 여러 룸 동시 전개 가능

**트릭룸**
- 5T; 속도 낮은 순서대로 행동
- 우선도 등은 그대로 적용

**매직룸**
- 5T; 모든 포켓몬의 소지품 효과 무효화 (서투름과 동일)
- 구애 상태는 해제되나 룸 종료 시 재구애
- 풍선/횃불 등 파괴 방지 효과도 미작동

**원더룸**
- 5T; 방어와 특방 실수치 교환
- 능력 상승/감소 변화는 이동 안 됨

### 17.3 필드 계열

**공통 특징**
- 5T (그라운드코트로 8T)
- 하나씩만 전개; 나중 전개가 덮어씀
- **지상에 있는 포켓몬에게만 효과**
  - 지상 아닌 포켓몬: 비행 타입, 부유 특성, 풍선, 전자부유/텔레키네시스 상태
  - 지상으로 강제: 검은철구, 중력 상태, 뿌리박기/떨어뜨리기 상태

**그래스필드**
- 매 턴 종료 시 최대HP 1/16 회복
- 풀 기술 ×1.5
- 지진/땅고르기/매그니튜드 ×0.5
- 보호색→풀, 자연의힘→에너지볼

**미스트필드**
- 상태이상/혼란 방지
- 드래곤 기술 ×0.5
- 타입 강화 없음 (유일)
- 보호색→페어리, 자연의힘→문포스

**일렉트릭필드**
- 전기 기술 ×1.5
- 잠듦/하품 방지 (기존 잠듦은 유지)
- 보호색→전기, 자연의힘→10만볼트

**사이코필드**
- 에스퍼 기술 ×1.5
- 우선도 +1 이상인 기술 차단 (단, 자신 대상이나 전체필드 기술은 차단 불가)
- 보호색→에스퍼, 자연의힘→사이코키네시스

### 17.4 기타 필드 상태

**중력**
- 5T; 공중 포켓몬도 압정/독압정/땅 기술 피격
- 특정 기술 사용 불가 (튀어오르기/전자부유/공중날기 등)
- 명중 ×1.67 (일격필살 기술 제외)
- 아침햇살 회복량 최대HP 1/4

**흙놀이**
- 5T; 전기 기술 위력 ×1/3 (아군 기술 포함)

**물놀이**
- 5T; 불꽃 기술 위력 ×1/3 (아군 기술 포함)

**페어리록**
- 2T; 필드 포켓몬 통상교대 불가 (실질 1T 효과)
- 아름다운허물/유턴 등으로는 회피 가능

**플라즈마샤워**
- 1T; 그 턴의 모든 노말 기술이 전기 타입으로 변환

## 18. 트레이너 자질

4가지 자질: 지시, 통솔, 육성, 능력  
랭크: E/D/C/B/A/AA/AAA/S (EX는 질적으로 독보적인 특수 케이스)

### 18.1 지시 자질

| 랭크 | 주요 효과 |
|------|-----------|
| C+ | 급소보정 C+1 (상대와 랭크 차이당) |
| A+ | 추가지령 (저격지령/회피지령), 혼성습득 (지시A+: 숙달/전개 혼합) |
| AA+ | 『맞춰라！』(1/시/선행: 아군 공격 필중; 4식과 조합 가능) |
| A+ | 묘수 (A 초과 랭크 수만큼 1/시합; 최적 기술 탐색) |

**지령 계열**
- 기본지령 (지시 E): 6가지 능력치 각 1/시합; 공격/방어/특공/특방/민첩/회복
  - 기본: 1T +1 랭크; 회복 = 턴 종료 시 +1/4 HP (선행)
- 숙달지령 (지시 C+): 능력 상승이 지속됨 (1T 아님); 회복지령 = 턴 시작 즉시 회복
- 전개지령 (지시 B+): 1T ×2배 강화; 회복 = 1/2 HP (다른 행동 포기)
- 지시 A+: 숙달+전개 혼합 가능

**4식 트레이너 포텐셜** (1/시합)
- 『물러나！』: 물리기술 1회 회피
- 『피해라！』: 특수기술 1회 회피
- 『돌아와！』: 자신 포켓몬 즉시 회수; 상대 기술 빗나감; 지정 포켓몬을 나중에 파견
- 『버텨라！』: 받는 데미지 절반 + 최소 HP1 유지 (1타만)
- 4식은 동시에 2개 사용 불가; 1개 선언 후 발동 안 돼도 사용으로 취급

### 18.2 통솔 자질

**기본 기능**
- 어떤 포켓몬을 영입할 수 있는지 결정 (필요통솔 E/D/C/B/A)

**추가역할 (A+부터)**

| 추가역할 | 효과 |
|----------|------|
| 톱배터 | 선발 시 임의 능력+2 |
| 선구의에이스 | 선발 시 전 능력+2 (1회, 교대 후 사라짐) |
| 양대간판 | 항상 전 능력×1.33 + C+1; 킬러 무효 |
| 체이서 | 자신 공격에 추격 효과 부여 |
| 인챈터 | 필드 퇴장 시 아군 임의 능력+1 |
| 스카우터 | 필드에 있는 동안 매 턴 1회 데이터 해석 가능 |
| 어벤저 | 죽어내밀기로 나올 때 임의 능력+2 |
| 엠퍼러 (AAA+) | 진입 시 랜덤 상대 능력 감소; 필드에 있는 동안 상대 전 능력 ×0.85 |

**특권포텐셜 — 익스펜션 (18종)**

| 타입 | 효과 |
|------|------|
| 노말 | 자속 ×2배 |
| 불꽃 | 얼음 면역 + 1할 추가 화상 |
| 물 | 턴 시작 시 중확률 상태이상 치유 |
| 풀 | 모래/싸라기눈 면역 + 충전 없이 기술 사용 |
| 전기 | 조이기 면역 + 1할 추가 마비 |
| 얼음 | 화상 면역 + 1할 추가 얼음 |
| 격투 | 방호포텐셜 무시 |
| 독 | 회복 → 데미지로 변환 |
| 땅 | 설치기술 면역 |
| 비행 | 날씨 디메리트 면역 + 1할 풀죽음 |
| 에스퍼 | 혼란 면역 + 1할 혼란 유발 |
| 벌레 | 회피 상승 무시 + 경험치↑ |
| 바위 | 방어 해제 불가 |
| 고스트 | 비공격 빈사 무효 |
| 드래곤 | HP 종족치 상승 |
| 악 | 추격 무효 |
| 강철 | 악/고스트에 특수 내성 |
| 페어리 | 상대 잠듦 무효 + 1할 잠듦 유발 |

**PT포텐셜**
- A: 1슬롯, AA: 2슬롯
- 파티 전원에게 포텐셜 부여

**유대포텐셜**
- 영혼의유대/사랑의유대 (기본 1개, A:2, AA:3, AAA:4, S:6+)
- 유대 포켓몬이 필드에 있는 동안 트레이너 지시+1랭크
- 왕의유대: 아군 포텐셜을 자신에게 적용 가능

**명중보정**
- A: 명중체감 정상화; AA: 명중+1; AAA: 명중+2; S: 명중+3

**야생 포텐셜**
- 야생포켓몬: 진입 시 임의 능력+2
- 야생의두목: 진입 시 전 능력+2

**필드효과 (《필드효과》)**
- 던전/이계/스타디움/짐 4종
- 통솔형 고유포텐셜로 생성 가능 (스타디움형, 홈 지정 포함)
- AAA+ 트레이너 필드는 하위 트레이너가 덮어쓸 수 없음

### 18.3 육성 자질

| 랭크 | 주요 효과 |
|------|-----------|
| C | 계제② 해금 |
| B | 계제③ 해금 |
| A | 계제④ 해금; 패배해도 경험치; A+: PT포텐셜 부여 |
| AA | 기술머신 추출; 선/회피/내성/격/기합 → 극화; 종족치 애매치 정상화 |
| AAA | 종족치 자체 증강 |

- 레벨 차이 10: 능력치 ×1.1 차이
- A+: 전투 없이 육성으로도 레벨 상승 가능

**기술머신**
- 육성 AA 이상이 포켓몬이 습득한 기술을 기술머신으로 만들 수 있음
- 기술머신(소지품): 1회만 해당 기술 사용 가능
- 실물은 남으므로 매 시합 재사용 가능

**기능확장**
- 기술목록 외 기술을 사용하는 행위 전반
- 고유포텐셜/기술머신/전용포텐셜/파티포텐셜로 내보낼 수 있음
- 구애 아이템 보유 시 기능확장으로 사용 → 구애 안 됨
- 단, 이미 구애된 상태에서 기능확장 사용 불가

### 18.4 능력 자질

| 랭크 | 고유포텐셜 | 전용포텐셜 |
|------|-----------|-----------|
| C | 1 | 3 |
| B | 1~2 | 3~5 |
| A | 2+ | 6 |
| AA | 2+ | 8 |
| AAA | - | 10 |

- 전용포텐셜: 포켓몬 1마리당 1개 (게임 설정상 최대 2개)
- 매우 강력한 효과 (삼각효과, 필중추미 등)

### 18.5 서포터 / 브리더

**서포터**
- 자신의 통솔 자질 -1랭크로 트레이너 통솔 보조
- 제한: 일부 추가역할만 임명 가능; 익스펜션/PT포텐셜 부여 불가; 유대 추가 불가
- 서포터 통솔 A+부터 추가역할 임명 가능 (서포터 고유 목록 내에서만)
- SP효과: 시합 중 응원으로 파티에 고유 능력 발동

**브리더**
- 자신의 육성 자질 -1랭크로 포켓몬 육성 보조
- 특기 분야(+판정)는 1랭크 저하 없이 제공 가능
- 육성 PT포텐셜 부여 가능

> **통솔/육성 자질은 외부 보완 가능** (서포터/브리더), 단 자체 랭크가 기반이어야 익스펜션/PT포텐셜 등 특권 가능
