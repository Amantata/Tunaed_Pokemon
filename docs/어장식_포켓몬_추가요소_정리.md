## 1. 포텐셜 시스템 개요

포텐셜은 포켓몬 배틀에서 특성·기술과 독립된 별도의 능력 체계입니다.

### 1.1 포텐셜 종류

**트레이너 포텐셜** (5종)
- 4식
- 지령 (6령)
- 데이터 해석
- 고유포텐셜
- 전용포텐셜

**포켓몬 포텐셜** (15종)

| 번호 | 포텐셜 | 필수 여부 |
|------|--------|-----------|
| 1 | 역할 | 임의 (역할별 조건) |
| 2 | 분류 | 임의 |
| 3 | 주인 | 임의 |
| 4 | 이명 | 조건부 취득 |
| 5 | 계제① | 필수 (Lv100+) |
| 6 | 계제② | 필수 (Lv150+) |
| 7 | 계제③ | 임의 (Lv200+) |
| 8 | 계제④ | 특수 조건 + 육성A+ |
| 9 | 속별 | **필수 (전 포켓몬)** |
| 10 | 유대 | 임의 |
| 11 | 선제 | **필수 (전 포켓몬)** |
| 12 | 회피 | **필수 (전 포켓몬)** |
| 13 | 내성 | **필수 (전 포켓몬)** |
| 14 | ○격 | **필수 (전 포켓몬)** |
| 15 | 범용 | **필수 (전 포켓몬)** |
| - | 부수 | 역할/범용 연결 강화 포텐셜 |
| - | 특권 | 통솔형 익스펜션 |
| - | PT① / PT② | 통솔/육성 부여 |
| - | 전용포텐셜 | 능력 자질로 부여 (별도 필드) |

> **필수 포텐셜**: 계제①~②, 속별, 선제, 회피, 내성, 격, 범용  
> **가변 포텐셜**: 역할, 분류, 주인, 이명, 계제③~④, 유대, 부수, 특권, PT①~②

## 2. 트레이너 포텐셜

### 2.1 『4식』

1/시합 사용; 선행 선언 또는 임의 사용 가능.

| 포텐셜 | 효과 |
|--------|------|
| 『물러나！』 | 물리기술 1회 회피 |
| 『피해라！』 | 특수기술 1회 회피 |
| 『돌아와！』 | 포켓몬 즉시 회수; 상대 기술 빗나감; 지정 포켓몬 나중에 파견 |
| 『버텨라！』 | 받는 데미지 ×1/2 + 최소 HP1 유지 (첫 타만) |

**제한 사항**
- 동시에 2개 사용 불가
- 선언 후 발동 안 돼도 사용으로 취급 (1회 차감)
- 『물러나！』/『피해라！』+ 지령 1개 동시 사용 가능

### 2.2 『지령』 (기본)

6가지 능력치 각 1/시합.

| 지령 | 기본 효과 | 숙달형 (C+) | 전개형 (B+) |
|------|-----------|-------------|-------------|
| 공격지령 | 공격+1 (1T) | 지속 | 1T ×2배 강화 |
| 방어지령 | 방어+1 (1T) | 지속 | 1T ×2배 강화 |
| 특공지령 | 특공+1 (1T) | 지속 | 1T ×2배 강화 |
| 특방지령 | 특방+1 (1T) | 지속 | 1T ×2배 강화 |
| 민첩지령 | 속도+1 (1T) | 지속 | 1T ×2배 강화 |
| 회복지령 | 턴 종료 시 +1/4 HP | 턴 시작 즉시 회복 | 1/2 HP 회복 (행동 포기) |

- 1턴에 지령 1개만 사용 가능
- 4식 + 지령 1개 조합 가능
- 지시 A+: 숙달지령 + 전개지령 혼합 가능

### 2.3 『데이터 해석』

- 『데이터 애널라이저』: 1/시합 — 상대 1체 데이터 분석

## 3. 포켓몬 포텐셜 — 속별

모든 포켓몬이 보유. 종/속 고유의 특성적 능력.

### 예시

| 포텐셜 | 효과 |
|--------|------|
| 『VOCALOID』 | 소리 기술이 방음 특성을 무시 |
| 『오라클세포』 | 상대 쓰러뜨리면 HP +1/4 회복 |
| 『우치하 일족』 | 보고흉내내기 우선도 +1 |

---

## 4. 포켓몬 포텐셜 — 선제

| 포텐셜 | 효과 |
|--------|------|
| 『선의 선』 | 상대가 우선도+1 이상 기술 사용 시 → 저확률로 자신이 먼저 행동 |
| 『대의 선』 | 같은 속도 티어에서 → 중확률로 우선도+1 |
| 『후의 선』 | 상대가 먼저 행동할 때 → 드물게 자신이 먼저 행동 |

**우선도 계층**: 가장 먼저 > 상대보다 먼저 > 우선도+N > 속도비교

## 5. 포켓몬 포텐셜 — 회피/내성/격

특정 타입 ○을 지정 (회피/내성/격은 같은 타입이어야 함).

| 포텐셜 | 효과 |
|--------|------|
| 『대○회피』 | 상대의 ○타입 기술 명중률 ×0.85 |
| 『대○내성』 | 상대의 ○타입 기술 데미지 ×0.67 |

**격 포텐셜 (5종 중 1개 선택)**

| 포텐셜 | 효과 |
|--------|------|
| 『대○저격』 | 저확률 → 자신 기술 필중 |
| 『대○추격』 | 저확률 → 자신 기술에 추격 효과 부여 |
| 『대○박격』 | 저확률 → 자신 기술 데미지 ×2 |
| 『대○반격』 | 저확률 → 턴 마지막에 추가행동 획득 |
| 『대○요격』 | 저확률 → 상대 기술 위력 절반 |

## 6. 포켓몬 포텐셜 — 범용

| 포텐셜 | 효과 |
|--------|------|
| 『기합』 | 드물게 빈사 기술을 버팀 (HP 가득 시 발동 어려움) |
| 『초근성』 | 중확률로 비공격 데미지 무시 |
| 『전투속행』 | 저확률로 받는 데미지를 자신 행동 이후로 지연; 유턴으로 퇴장 시 지연 데미지도 함께 적용 |
| 『진지작성』 | 자신이 전개하는 필드 기술 지속 +1T (기술만, 특성 전개는 제외) |
| 『군략』 | 자신의 전체기술 ×1.2; 상대의 전체기술 ×0.8 |

## 7. 포켓몬 포텐셜 — 역할

### 7.1 기본 4종

| 역할 | 발동 조건 | 효과 |
|------|-----------|------|
| 『선발』 | 먼저 필드에 나올 때 | 전 능력 ×1.1 |
| 『에이스』 | 「여기다！」 발동 (아군 ≤50% 잔존 OR 양측 에이스 대결) | 전 능력 ×1.33 + C+1 (1회 지속 발동) |
| 『에이스 어시스트』 | 에이스를 교대로 내보낼 때 | 8가지 능력치 중 1개 +1 |
| 『에이스 킬러』 | 에이스 포켓몬에 대한 공격 | 데미지 ×1.5 |
| 무직 | — | 역할 없음 |

### 7.2 선발 발전형 8종

기본 효과: 전 능력 ×1.1 → 특정 1개 능력 +1로 변환

귀인, 경화, 마술사, 현자, 광주, 매의눈, 심안, 사냥꾼

**선발 부수**
- 약: 방어 성공 시 대응 능력+1; 방어 미사용 시 스타터 (전 능력 ×1.05)

### 7.3 어시스트 발전형

| 형태 | 효과 |
|------|------|
| α | 첫 기술 위력 ×2 |
| β | 첫 피격 전까지, 또는 첫 행동 전까지 데미지 절반 |
| γ | 어떤 상황으로든 퇴장 시 능력+1 |

**어시스트 부수**: 리턴힐/리턴큐어 (임의교대 시 중확률 발동)

### 7.4 킬러 발전형

| 형태 | 효과 |
|------|------|
| α | 에이스에게 ×1.5 데미지 + 상대 방어/특방 상승 무시 |
| β | 에이스로부터 받는 데미지 절반 + 상대 공격/특공 상승 무시 |
| γ | 에이스가 필드에 있을 때 속도 ×2 + 상대 속도/회피 상승 무시 |

**킬러 부수**
- 챌린지: 턴 종료 시 자신 우선도 +1
- 앰부시: 상대 에이스의 첫 기술 우선도를 -1로 설정

### 7.5 에이스 발전형

- 부동의에이스: 전 능력 ×1.5 + C+1 (기본 에이스의 강화판)

**에이스 부수**
- 카운트아츠: 회피율 ×1.33 (킬러 상대 시 비활성)
- 슬러그아츠: 명중률 ×1.33 (킬러 상대 시 비활성)

## 8. 포켓몬 포텐셜 — 분류

### 8.1 고대종계

| 포텐셜 | 효과 |
|--------|------|
| 『올드타입』 | 비올드타입의 방호포텐셜 무시 (틀깨기와 유사) |
| 『적응종』 | 올드타입에 면역 |
| 『델타종』 | 필드에 있는 동안 상대 올드타입 무효화 |
| 『메가진화』 | 올드타입 취급 + 상대 올드타입 무효화 |
| 『원시천부』 | 에이스 발동 시 트레이너 통솔+1 |

### 8.2 고유종계

| 포텐셜 | 효과 |
|--------|------|
| 『고유종』 | 하나뿐인 개체 → 올드타입 면역 |
| 『천부의재능』 | 최고의 개체 → 에이스 발동 시 트레이너 통솔+1 |
| 『금관사이즈』 | 최대 크기 → HP 대폭 상승 |
| 『유체종』 | 사후 고스트로 부활; 노말/격투 면역 (타입 기반 아님; 배짱/틀깨기로 무효화 불가) |

### 8.3 변종계

| 포텐셜 | 효과 |
|--------|------|
| 『변종』 | 다른 생태 → 올드타입 면역 |
| 『하프』 | 포켓몬×인간 혼혈 → 자타입 기술 데미지 ×0.5 |
| 『혈계천부』 | 하프 + 천부의재능 효과 |

## 9. 방호포텐셜 분류

올드타입이 관통하는 방어 계열 포텐셜의 분류.

| 분류 | 올드타입 관통 여부 |
|------|-------------------|
| 무효(반사)계 | ✅ 관통 |
| 데미지경감계 | ✅ 관통 |
| 버티기계 | ✅ 관통 |
| 내구강화계 | ✅ 관통 |
| 방어종족치변화계 | ⚠️ 논란 있음 |
| 급소무효계 | ✅ 관통 |
| 상승(강화)무시계 | ✅ 관통 |
| 상성변경계 | ✅ 관통 |
| 회피율강화계 | ❌ 관통 안 됨 |
| 상태변화/이상내성계 | ❌ 관통 안 됨 |

**급소 관통 정책 (설정 가능)**
- 전부관통파 / 완화계까지만파 / 비관통파
- 이 게임 기본값: **완화계까지만** 관통

## 10. 포켓몬 포텐셜 — 종족 (계제)

속별보다 개인적/전술적 특성을 반영.

| 계제 | 해금 조건 |
|------|-----------|
| 계제① | Lv 100 이상 |
| 계제② | Lv 150 이상 |
| 계제③ | Lv 200 이상 |
| 계제④ | 특수 조건 + 육성자질 A 이상 |

### 예시

- 『가희요정의환주』: 「여기다！」 발동 시 페어리 기술 필중
- 『가희요정의열창』: 소리 기술이 방어/대타출동을 관통

## 11. 포켓몬 포텐셜 — 극화 (육성 AA 이상)

육성 자질 AA 이상의 트레이너만 부여 가능.

### 극화 포텐셜 목록

| 포텐셜 | 효과 (기존 대비 강화) |
|--------|----------------------|
| 『삼종의 신기』 | 선의 선/대의 선/후의 선 모두 보유 |
| 『극○회피』 | 저확률 → 실제 기술 회피 (기존 회피: 명중률 감소만) |
| 『극○내성』 | 데미지 반감 (기존: ×0.67) |
| 『극○저격』 | 중확률 필중 (기존: 저확률) |
| 『극○추격』 | 중확률 추격 (기존: 저확률) |
| 『극○박격』 | 중확률 데미지 ×2 (기존: 저확률) |
| 『극○반격』 | 중확률 추가행동 (기존: 저확률) |
| 『초기합』 | 드물게 버티기 + 중확률로 비공격 데미지 무시 |

> ⚠️ 극화 포텐셜은 상대 타입과 무관하게 지정 타입 기술이면 반응 — 기존 회피/내성보다 훨씬 강력함

## 12. 고유포텐셜 / 전용포텐셜

### 고유포텐셜 (트레이너 고유)

- 트레이너마다 고유; 파티 전원에게 영향
- 능력 자질이 높을수록 수가 늘어남 (C=1, B=1~2, A=2+, AA=2+, AAA=많음)
- 예: 파티가 특정 기술 사용 가능, 전 기술 급소율 상승

### 전용포텐셜 (포켓몬 전용)

- 트레이너의 능력 자질로 포켓몬에게 부여
- 포켓몬 1마리당 1개 (게임 설정 최대 2개)
- 매우 강력한 효과 (삼각효과, 필중추미 등 예시)
- 잃으면 재습득이 매우 어려움
- 도감 분석으로는 알 수 없는 숨겨진 능력

> **명칭 엄수**: 고유포텐셜 = 트레이너 고유 / 전용포텐셜 = 포켓몬 전용 (절대 혼용 금지)

---

## 13. 특수 역할 포텐셜

### 13.1 『타이트 엔드』

통솔 자질과 무관; 포켓몬의 적성이 있어야 취득 가능. 적성은 희귀.

| 형태 | 구성 역할 |
|------|-----------|
| 타이트 엔드 (기본) | 선발 + 킬러 + 어시스트 |
| 타이트 엔드α | 선발 + 킬러α + 어시스트α |
| 타이트 엔드β | 선발 + 킬러β + 어시스트β |
| 타이트 엔드γ | 선발 + 킬러γ + 어시스트γ |

- 성향에 따라 α/β/γ 중 하나로만 발전
- 다른 선발/킬러/어시스트와 중복 가능

**타이트 엔드 부수**

| 포텐셜 | 효과 |
|--------|------|
| 『조커』 | 상대와 같은 역할일 때 기술 데미지 ×1.5 |
| 『제스터』 | 타이트 엔드 임명 상태에서 스타터/챌린지/리턴큐어로 취급 |

### 13.2 『베테랑』

선수생명 한계에 도달한 포켓몬이 취득. 역할이므로 통솔 자질과 무관.

**특징 3가지**

1. **독자적 전용포텐셜**: 트레이너 능력 자질 무관; 이전 경기 경험을 포텐셜화; 개인 성향 범위 내에서만
2. **노쇠**: 속별 포텐셜이 디메리트로 변화; 능력치 미세 저하; 계제④/극화 달성 어려움
3. **이명**: 개체 의존 이명을 반드시 보유 (이명 수량 제한에 포함 안 됨)

**베테랑 부수**

| 포텐셜 | 효과 |
|--------|------|
| 『어드바이스』 | 타임 사용 시 상대 데이터 해석 가능 |

## 14. 이명 포텐셜

큰 활약이 사람들의 기억에 남아 얻는 포텐셜.

### 14.1 종류

| 종류 | 조건 | 효과 유지 |
|------|------|-----------|
| 트레이너 의존 | 특정 트레이너 아래에서 활약 | 해당 트레이너 지휘 하에서만 발동 |
| 개체 의존 | 특정 활약 달성 | 트레이너 무관하게 발동 |

**예시**
- 『모략』: 메리가 트레이너일 때, PT 참가 중 트레이너 지시+1랭크 (트레이너 의존)
- 『악식』: 난동부리기 상태일 때, 매 턴 개시 시 HP +1/6 회복 (개체 의존)

### 14.2 이명 발동 수 제한

| 상황 | 동시 발동 가능 이명 수 |
|------|----------------------|
| 일반 | 3체 |
| 리그 A랭크 | 4체 |
| 리그 S랭크 | 6체 (전원) |

- 베테랑의 이명은 이 제한에 포함되지 않음

### 14.3 인지도 시스템

- 이명 한계 초과 시 인지도가 높은 쪽이 발동
- 전설종과 유사한 현상으로 연구 대상

## 15. 서포터와 브리더

### 15.1 서포터

트레이너의 통솔을 보조하는 역할.

**기본 효과**: 자신의 통솔 자질 -1랭크 수준을 트레이너에게 제공

**제한 사항**
- 임명할 수 있는 추가역할은 일부만 (서포터마다 고유 목록)
- 통솔형 특권(익스펜션, PT포텐셜) 부여 불가
- A이상이어도 유대 추가 불가

**통솔 산정 규칙 (복잡 케이스)**

| 상황 | 가능 여부 |
|------|-----------|
| 자체통솔A → 천부로 AA | PT포텐셜 2개째 부여 가능 |
| 자체통솔B + 천부로 A | 추가역할/유대 가능, PT/익스펜션 불가 |
| 서포터 첨부 통솔 A | 서포터 고유 추가역할만, 기타 통솔 혜택 없음 |

**SP효과**: 시합 중 응원으로 파티 전체에 고유 능력 발동

### 15.2 브리더

포켓몬 육성을 보조하는 역할.

**기본 효과**: 자신의 육성 자질 -1랭크 수준으로 포켓몬 육성

**특이사항**
- 특기 분야(+판정)는 1랭크 저하 없이 제공 가능
- 육성을 통한 PT포텐셜 부여 가능
- 반드시 자체 트레이너가 육성A 이상이어야 PT포텐셜/익스펜션 부여 가능

> ⚠️ **계약**: 지시/능력 자질 보완 수단으로 "계약"이 존재하나, 파멸적 결과를 수반하므로 설명 생략 (스스로 조사하도록 안내됨)

## 16. 턴 중 교대 관련 규칙

### 16.1 기본 원칙

교대로 필드에 나온 포켓몬은 **그 턴에 행동권 없음** (기본 원칙).

### 16.2 예외 ①: 전용포텐셜

전용포텐셜에 "통상교대로 나온 가장 마지막에도 발동할 수 있다"는 명시가 있는 경우.

### 16.3 예외 ②: 컨셉용 변화기술

목적 자체가 상태변화 유발이고, 컨셉을 위해 특정 변화기술로 표현된 경우.

```
예: 아군의 역할 발동 시, 아군이 「느림보씨」를 내보낸다
  → 실제로는 상대 특성을 「시간벌기」로 바꾸는 것이 목적
  → 필드에 나오자마자 교대로 나온 턴에도 발동 가능
```

### 16.4 T종료 시 포텐셜 효과

- 교대로 나온 포켓몬에게는 T종료 시 효과가 발생하지 않음
- 단 "필드에 나온 T 종료 시"의 표현은 **선발/죽어내밀기/T종료 교대로 나온 경우에만** 적용되는 제약

### 16.5 소지품 T종료 효과

- 먹다남은음식/화상 등: 교대 나온 턴 T종료 효과 없음
- **예외**: 화염구슬/맹독구슬은 교대 나온 턴에도 T종료 시 상태이상 적용

## 부록: 포텐셜 명명 규칙 (PT-05)

- **고유포텐셜**: 각 트레이너만의 고유 포텐셜. 트레이너마다 다름.
- **전용포텐셜**: 포켓몬 전용의 강력한 포텐셜. 도감 분석으로 알 수 없음.
- 이 두 개념은 **절대 혼동해서는 안 되며**, 데이터 모델에서도 별도 필드로 관리.
