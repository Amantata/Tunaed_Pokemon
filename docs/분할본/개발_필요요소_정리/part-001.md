## 2. 배틀 엔진 — 기본 시스템

### 2.1 배틀 상태 스냅샷 (BattleStateSnapshot)

- [ ] 전체 배틀 상태를 JSON으로 직렬화/역직렬화
- [ ] 턴 히스토리 리스트 (Undo/Redo용)
- [ ] 이벤트 로그 (애니메이션 재생용)

### 2.2 턴 진행 파이프라인

- [ ] 1단계: 턴 시작 처리 (T개시 효과)
- [ ] 2단계: 커맨드 입력 (4식/지령 선언 포함)
- [ ] 3단계: 행동 순서 결정
  - [ ] 가장 먼저 > 상대보다 먼저 > 우선도+N > 속도 비교
  - [ ] 트릭룸 상태 시 속도 역전
  - [ ] 마비/능력변화 적용 후 속도 계산
- [ ] 4단계: 기술/교대 실행
- [ ] 5단계: 턴 종료 처리
- [ ] 6단계: 상태이상/날씨 데미지 → 드레인 → 회복 → 교대

### 2.3 데미지 계산

- [ ] 물리/특수 분류별 공/방 vs 특공/특방 계산
- [ ] 강화(배율) 적용 (구애머리띠, 생명의구슬 등)
- [ ] 상승(랭크) 적용 (능력변화 랭크 배율표)
- [ ] STAB (자속보정) ×1.5
- [ ] 타입 상성 (18×18 타입 차트)
  - [ ] 이중 타입 포켓몬: 두 타입 상성 곱산
  - [ ] 이중 타입 기술: 더 유리한 상성으로 계산
- [ ] 급소 처리: 확률/배율/공격자 감소 무시/방어자 상승 무시/리플렉터 관통
- [ ] 랜덤 변동 (0.85~1.0)
- [ ] 사이코쇼크형 (특공 vs 방어) 특수 처리
- [ ] 고정 데미지 기술 (일격필살, 용성군 등)

### 2.4 명중/회피 판정

- [ ] 명중율 계산: 기술 명중률 × (명중 랭크 배율) / (회피 랭크 배율)
- [ ] 필중 효과 처리 (마음의눈, 텔레키네시스, 중력 등)
- [ ] 트레이너 자질 명중보정 (통솔A: 정상화, AA: +1, AAA: +2, S: +3)

### 2.5 급소 (크리티컬) 계산

- [ ] 기본 1/16; +1=1/8; +2=1/4; +3=1/3; +4=1/2
- [ ] 주술 상태: 급소 완전 차단
- [ ] 올드타입 급소무효 관통 여부 (설정값 기반)

### 2.6 교대 시스템

- [ ] 임의교대 (통상교대 포함)
- [ ] 강제교대 (레드카드, 날려버리기 등)
- [ ] 유턴/바톤터치 교대
- [ ] 죽어내밀기
- [ ] 추격 효과 (체이서/추격 등)
- [ ] 교대 불가 조건 체크 (조이기/검은눈빛/뿌리박기/페어리록)
- [ ] 교대 시 능력변화 초기화 (조이기, 장기전 효과, 구애 등 예외)

### 2.7 행동 순서 / 우선도

- [ ] 우선도 -7 ~ +7 관리
- [ ] 동일 우선도 내 속도 비교
- [ ] 트릭룸, 습지초원(속도 1/4) 상호작용
- [ ] 선제 포텐셜 효과 (가장 먼저 / 상대보다 먼저)

## 3. 배틀 엔진 — 상태이상 / 상태변화

### 3.1 상태이상 (6종)

- [ ] 독: 매 T종료 1/8 HP 데미지
- [ ] 맹독: 누적 증가 데미지 (교대 시 리셋)
- [ ] 얼음: 행동 불가 (1/5 자연해제, 쾌청/불꽃기술 해제)
- [ ] 마비: 속도 ×1/2; 1/4 확률 행동 불가
- [ ] 잠듦: 1~4T; 2체 동시 잠듦 한계
- [ ] 화상: 매 T종료 1/8 HP; 물리 공격력 절반
- [ ] 혼란: 별도 관리 (자해 판정)

### 3.2 상대에게 적용되는 상태변화

- [ ] 혼란 (1~4T, 자해, 접촉 무관)
- [ ] 풀죽음 (행동 스킵, 멘탈효과)
- [ ] 저주 (고스트형 / 비고스트형)
- [ ] 씨뿌리기 (매 T 1/8 흡수; 풀타입 면역)
- [ ] 마음의눈 (필중; 잠복상태 타격; 사용자 퇴장 시 해제)
- [ ] 앵콜 (멘탈효과)
- [ ] 도발 (멘탈효과)
- [ ] 검은눈빛 (통상교대 불가; 고스트 면역)
- [ ] 하품/동면 (T종료 시 잠듦/얼음 유발)
- [ ] 회복봉인 (멘탈효과)
- [ ] 사슬묶기 (멘탈효과)
- [ ] 봉인 (교대해도 유지)
- [ ] 떨어뜨리기 (지면화)
- [ ] 텔레키네시스 (공중; 필중; 교대 유지)
- [ ] 송전/악인/침수 (타입 변환 1T)
- [ ] 미래예지/파멸의소원 (지연 데미지)
- [ ] 폭탄 (접촉 반격)
- [ ] 조이기 (통상교대 불가; 매 T 1/8 데미지; 사용자 퇴장 후에도 유지)

### 3.3 자신에게 적용되는 상태변화

- [ ] 대타출동 (HP 1/4 소비; 차단/비차단 목록 관리)
- [ ] 방어 (+4 우선도; 예외 목록 관리)
- [ ] 길동무 (공격받아 빈사 시 상대도 빈사)
- [ ] 매직코트 (변화기술 반사)
- [ ] 공중날기/구멍파기/다이빙/섀도다이브 (잠복 상태)
- [ ] 작아지기 (특정 기술 취약)
- [ ] 웅크리기 (구르기/아이스볼 강화)
- [ ] 분노 (공격받을 때마다 공격+1)
- [ ] 뿌리박기 (+1/16 HP; 강제교대 면역; 통상교대 불가; 땅기술 피격)
- [ ] 비축하기 (1~3중첩)
- [ ] 참기 (2T 행동불가; 반격)
- [ ] 날따름 (단일 기술 유인)
- [ ] 파워트릭 (공격/방어 교환)
- [ ] 난동부리기 (2T 강제; 종료 시 혼란)
- [ ] 소란피기 (난동 유사; 잠듦 방지)
- [ ] 아쿠아링 (매 T 1/8 HP 회복)
- [ ] 충전/냉각/가열 (다음 기술 ×2)
- [ ] 전자부유 (공중; 교대/피격으로 해제)

### 3.4 멘탈효과 일괄 처리

멘탈효과: 헤롱헤롱, 앵콜, 트집, 사슬묶기, 도발, 회복봉인  
- [ ] 멘탈허브 소지품으로 일괄 해제
- [ ] 마이페이스 특성: 혼란 면역

## 4. 배틀 엔진 — 필드 환경

### 4.1 자진 필드 효과

- [ ] 순풍 (속도 ×2, 4T)
- [ ] 리플렉터 (물리 반감, 5T)
- [ ] 빛의장막 (특수 반감, 5T)
- [ ] 오로라베일 (물리+특수 반감, 싸라기눈 조건, 5T)
- [ ] 신비의부적 (상태이상 차단, 5T, 예외 목록)
- [ ] 주술 (급소 차단, 5T)
- [ ] 흰안개 (능력감소 차단, 5T)
- [ ] 무지개 (추가효과율 ×2, 더블배틀)

### 4.2 적진 설치기술

**대지설치기술**
- [ ] 압정뿌리기 (3중첩 지원; 공중 면역)
- [ ] 독압정 (2중첩; 독타입 흡수; 미스트필드 상호작용)
- [ ] 끈적끈적네트 (속도-1)

**타입설치기술** (상호 덮어쓰기)
- [ ] 스텔스록 (상성/8 데미지)
- [ ] 습지초원 (속도×1/4, 더블배틀)
- [ ] 불바다 (1/8 HP 데미지/턴, 더블배틀)

### 4.3 날씨

- [ ] 쾌청 (불꽃×1.5, 물×0.5, 얼음 불가, 솔라빔 즉시, 등)
- [ ] 큰가뭄 (쾌청+물 무효, 지속형)
- [ ] 비 (물×1.5, 불꽃×0.5, 번개 필중, 등)
- [ ] 폭우 (비+불꽃 무효, 지속형)
- [ ] 싸라기눈 (비얼음 1/16 데미지/턴, 눈보라 필중)
- [ ] 모래바람 (바위특방×1.5, 비바위강철 외 1/16 데미지/턴)
- [ ] 난기류 (비행 약점 완화, 지속형)
- [ ] 웨더볼 타입 변환 지원 (날씨별)
- [ ] 날씨 지속 +3T 아이템 지원 (뜨거운/보송보송/차가운/축축한바위)

### 4.4 룸 계열

- [ ] 트릭룸 (속도 역전, 5T, 재사용으로 해제)
- [ ] 매직룸 (아이템 무효, 5T)
- [ ] 원더룸 (방어/특방 교환, 5T)

### 4.5 필드 계열 (Terrain)

> ⚠️ 「필드」(terrain)와 《필드효과》(arena/stadium)는 완전히 다른 개념 (FE-06 주의)

- [ ] 그래스필드 (+1/16 HP/턴, 풀×1.5, 지진 반감, 5T)
