## 9. 파악된 규칙 예외 사항 정리

개발 시 특히 주의가 필요한 예외/특이 사항 목록.

### 9.1 교대 관련

| 예외 사항 | 처리 방법 |
|-----------|-----------|
| 화염구슬/맹독구슬 → 교대 나온 턴에도 T종료 적용 | 소지품 타입별 플래그로 구분 |
| 먹다남은음식/상태이상 데미지 → 교대 나온 턴 미적용 | 기본 처리 |
| 독 타입 + 독압정 → 독압정 제거 (땅에 있을 때만) | 타입 + 지상 여부 체크 |
| 미스트메이커 진입 → 독압정보다 먼저 처리 | 진입 효과 → 설치물 순서 |
| 뿌리박기 바톤터치 → 방/특방 상승 유지, 뿌리박기 상태 미인계 | |
| 유체종 → 노말/격투 면역, 배짱/틀깨기로 무효화 불가 | 별도 플래그 |

### 9.2 필드 상태 상호작용

| 예외 사항 | 처리 방법 |
|-----------|-----------|
| 안개제거: 리플렉터/빛의장막/오로라베일/신비의부적/흰안개만 제거 (순풍/주술 제외) | 제거 대상 목록 관리 |
| 리플렉터: 사이코쇼크형 (특공 vs 방어) 기술에는 효과 없음 | 기술 플래그 체크 |
| 혼란 자해 데미지: 리플렉터/오로라베일에 반감 안 됨 | 자해 판정 별도 처리 |
| 싸라기눈/모래바람: 대타출동/공중날기/방어 중에도 피격 | 날씨 데미지는 상태 무관 |
| 미스트필드: 잠자기(자신이 사용) 막지 않음 | 자신 대상 예외 |
| 일렉트릭필드: 기존 잠듦 상태는 깨우지 않음 | 신규 적용만 차단 |
| 사이코필드: 자신 대상/전체필드 기술의 우선도는 차단 불가 | 대상 타입 체크 |
| 중력: 아침햇살 등 회복량 1/4로 감소 (쾌청 동시 전개 시 2/3) | 중력+쾌청 우선순위 |
| 그래스필드: 지진/땅고르기/매그니튜드만 반감 (모든 땅 기술 아님) | 기술 플래그 체크 |

### 9.3 특성 vs 포텐셜

| 예외 사항 | 처리 방법 |
|-----------|-----------|
| 특성 변화 기술은 포텐셜에 영향 없음; 포텐셜 변화도 특성에 영향 없음 | 별도 해소 스택 |
| 올드타입: 회피율 강화계/상태이상 내성계 방호포텐셜은 관통 불가 | 분류 태그 관리 |

### 9.4 기타

| 예외 사항 | 처리 방법 |
|-----------|-----------|
| 일격필살: 레벨 100의 자리 수 기준 비교 | 레벨 체크 로직 |
| 일격필살: 기합의띠로 버티기 가능 | 고정 데미지 플래그 |
| 봉인: 교대해도 유지 (대부분의 상태변화는 교대 해제) | 영속성 플래그 |
| 난동부리기: 빗나가거나 풀죽음 시 혼란 없음 (정상 종료 시에만) | 혼란 조건 체크 |
| 트릭룸: 능력변화/마비로 감소된 속도 기준으로 역전 계산 | 실제 전투 속도 기준 |
| 매직룸: 구애 해제 후 룸 종료 시 재구애 | 구애 상태 영속 관리 |
| 날따름 + 선취: 선취 대상은 날따름 무효나 결과 기술은 날따름 포켓몬 대상이 됨 | 우선도/대상 복합 처리 |
| 교대로 나온 턴 행동권: 기본 없음 (전용포텐셜 명시 또는 컨셉형 변화기술은 예외) | 행동권 플래그 |
| 통솔/서포터/천부의재능 복합 계산: 특권(PT/익스펜션)은 자체 랭크 A 이상이어야만 | 자체 랭크 별도 추적 |
