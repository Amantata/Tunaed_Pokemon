## 1. 포켓몬 데이터 구조

포켓몬 데이터는 6개 항목으로 구성됩니다.

| 번호 | 항목 | 설명 |
|------|------|------|
| ① | 이름 | 포켓몬 이름 (별명 포함 가능) |
| ② | 타입 | 최대 2가지 타입 |
| ③ | 특성 | 고유한 패시브 효과 |
| ④ | 소지품 | 들려준 아이템 |
| ⑤ | 기술목록 | 전투에 사용할 수 있는 기술 목록 |
| ⑥ | 능력치 | 6종 스탯의 종족치 랭크 (E/D/C/B/A/AA/AAA/S, +/- 포함) |

## 2. 타입 시스템

### 2.1 18가지 타입

노말, 불꽃, 물, 전기, 풀, 얼음, 격투, 독, 땅, 비행, 에스퍼, 벌레, 바위, 고스트, 드래곤, 악, 강철, 페어리

### 2.2 STAB (자속보정)

같은 타입의 기술을 사용할 경우 위력 ×1.5

### 2.3 타입 면역 — 상태이상

| 타입 | 면역 대상 |
|------|-----------|
| 불꽃 | 화상 |
| 얼음 | 얼음 (상태이상) |
| 전기 | 마비 |
| 풀 | 씨뿌리기 |
| 독, 강철 | 독·맹독 |
| 고스트 | 통상교대 불가 |

### 2.4 타입 상성표 (18×18)

풀→물→불꽃→풀, 격투→바위→비행→격투 등 기본 상성 관계. 게임 원작의 18타입 상성표 적용.

## 3. 특성 시스템

각 포켓몬 고유의 패시브 능력. 배틀 중 자동 발동.

### 주요 특성 예시

| 특성명 | 효과 |
|--------|------|
| 가희 | 소리 기술 위력 ×1.5 |
| 불꽃몸 | 접촉 기술 받으면 30% 확률로 화상 |
| 잔비 | 필드에 나오면 비 날씨 발생 |
| 매직미러 | 상태변화 기술을 반사 |
| 우격다짐 | 추가효과 없음 대신 위력 ×1.3 |
| 초식 | 풀 기술 흡수 → 공격+1 |
| 가속 | 매 턴 종료 시 속도+1 |
| 프레셔 | 필드 진입 시 상대 속도-1 |
| 하늘의은총 | 기술 추가효과 발생률 ×2 |
| 부유 | 공중에 떠있어 땅 기술 무효 |
| 에어록, 날씨부정 | 날씨 효과 무효화 |
| 매직가드 | 직접 공격 이외 피해 무효 |
| 끝의대지 | 큰가뭄 발생 |
| 델타스트림 | 난기류 발생 |
| 미스트메이커 | 필드 진입 시 미스트필드 전개 |
| 지구력 | 공격받을 때마다 방어+1 |

## 4. 기술 시스템

### 4.1 기술 데이터 형식

```
기술명… 분류/타입/위력/명중/대상/우선도/접촉여부/효과
```

### 4.2 분류

| 분류 | 데미지 계산 방식 |
|------|----------------|
| 물리 | 공격 vs 방어 |
| 특수 | 특공 vs 특방 |
| 변화 | 데미지 없음, 효과만 |

### 4.3 대상 종류

자신, 단일, 상대전체, 자신이외, 전체필드, 아군필드, 상대필드

### 4.4 우선도

| 우선도 | 기술 예시 |
|--------|-----------|
| +4 | 방어, 판별, 킹실드, 니들가드 |
| +3 | 속이다 |
| +2 | 신속 |
| +1 | 아쿠아제트 |
| ±0 | 일반 기술 |
| -7 | 한판뒤집기 |

### 4.5 접촉 기술의 특이사항

- 접촉 기술은 상대의 특성(불꽃몸 등)이나 소지품(울퉁불퉁멧 등)을 트리거할 수 있음
- 방호패드 소지 시 접촉 트리거 무효

### 4.6 이중 타입 기술

일부 기술은 2가지 타입을 가짐 (예: 플라잉프레스 = 격투/비행). 더 유리한 상성으로 계산하며 양쪽 효과 모두 적용.

### 4.7 일격필살 기술

- 명중 시 상대 남은 HP만큼 고정 데미지
- 명중률 30 고정; 명중/회피 상승 무효 (단, 노가드/마음의눈 상태는 예외)
- 사용자 레벨 ≥ 상대 레벨 이상이어야 효과 발동
- 기합의띠/버티기로 버티기 가능; 타입 0배 시 무효
- 타입 약화 열매 발동 불가 (고정 데미지이므로)

## 5. 능력치 시스템

### 5.1 6대 능력치

HP, 공격, 방어, 특공, 특방, 속도

### 5.2 종족치 랭크

| 랭크 | 수치 범위 |
|------|-----------|
| S | 200+ |
| AAA | 170~195 |
| AA | 140~165 |
| A | 111~135 |
| B | 90~105 |
| C | 60~85 |
| D | 40~55 |
| E | 1~35 |

- 각 랭크에 +/- 보정 존재 (A+ = 135, A = 111~134, A- = 110)
- HP는 데이터 시트에 미표시 (특별 취급)
- 개체치(IV) 존재, 노력치(EV)는 기본 0

## 6. 능력변화 (랭크 시스템)

### 6.1 변화 가능 능력치 7종

공격, 방어, 특공, 특방, 속도, 명중, 회피

### 6.2 전투 능력치 변화 배율

| 랭크 변화 | 배율 |
|-----------|------|
| +n | (2+n)/2× |
| -n | 2/(2+n)× |

### 6.3 명중/회피 변화 배율

| 랭크 변화 | 배율 |
|-----------|------|
| +n | (3+n)/3× |
| -n | 3/(3+n)× |

- 교대 시 전부 초기화

### 6.4 강화 vs 상승 구분 (ST-02)

- **강화**: 직접 배율 적용 (×2 등), 랭크 단계 무관
- **상승**: 랭크 단계 변화 (+1, +2 등)
- 데미지 공식에서 별도 처리됨

## 7. 급소 (크리티컬)

### 7.1 확률

| 단계 | 확률 |
|------|------|
| 기본 | 1/16 |
| +1 | 1/8 |
| +2 | 1/4 |
| +3 | 1/3 |
| +4 | 1/2 |

### 7.2 효과

- 데미지 ×2 (일부 ×3)
- 공격자의 능력 감소 무시
- 방어자의 능력 상승 무시
- 리플렉터/빛의장막 관통
- 급소는 능력변화로 취급하지 않음

## 8. 상태이상

6종 + 혼란 (실질 7종)

| 상태이상 | 효과 |
|----------|------|
| 독 | 매 턴 종료 시 최대HP 1/8 데미지 |
| 맹독 | 시작 1/16, 매 턴 1/16씩 증가; 교대 시 리셋 |
| 얼음 | 행동 불가; 쾌청/불꽃기술로 해제; 매 행동 1/5 확률 자연해제 |
| 마비 | 속도 ×1/2; 매 행동 1/4 확률로 행동 불가 |
| 잠듦 | 1~4T 행동 불가 (잠자기: 2T); 동시에 양측 2체 이상 잠듦 불가 (2체 잠듦 규칙) |
| 화상 | 매 턴 종료 시 최대HP 1/8 데미지; 물리 공격력 절반 |
| 혼란 | 1~4T; 1/2 확률 자해 (위력 40 물리; 공격 랭크 적용; 리플렉터 무관) |

- 중첩 불가 (동시에 하나만)
- 교대해도 유지 (혼란 제외)

## 9. 상태변화 — 상대에게 적용

### 혼란

- 1~4T 지속; 1/2 확률 자해 (위력 40, 공격 랭크 적용)
- 구애머리띠/테크니션/불가사의부적 영향 없음

### 풀죽음

- 그 턴 남은 행동 스킵
- 멘탈효과

### 저주 (고스트 타입)

- 자신 HP 1/2 감소 → 상대에게 저주 부여 → 상대 매 턴 종료 시 1/4 HP 데미지

### 저주 (비고스트 타입)

- 속도-1, 공격/방어+1

### 씨뿌리기

- 매 턴 종료 시 1/8 HP 흡수당함
- 풀 타입 면역 (이미 풀 타입이 돼도 해제 안 됨)

### 마음의눈

- 상대가 받는 다음 기술이 필중
- 공중날기/구멍파기/다이빙/섀도다이브 상태에서도 적중
- 사용자가 교대하면 효과 사라짐

### 앵콜

- 마지막에 사용한 기술만 사용 가능
- 멘탈효과

### 도발

- 변화기술 사용 불가
- 멘탈효과

### 검은눈빛

- 통상교대 불가
- 사용자가 필드를 나가도 유지
- 고스트 타입 면역
- 유턴/바톤터치/탈출버튼으로는 교대 가능

**멘탈효과 목록**: 헤롱헤롱, 앵콜, 트집, 사슬묶기, 도발, 회복봉인
