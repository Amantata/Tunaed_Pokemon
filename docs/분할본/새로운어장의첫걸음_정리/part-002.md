
- 최대HP 1/4 소비하여 분신 생성; 분신이 대신 데미지 수신
- **차단되는 효과**: 혼란/씨뿌리기/하품/악몽/조이기/마음의눈/금제/회복봉인/특성없음/텔레키네시스/검은눈빛
- **차단 안 되는 효과**: 헤롱헤롱/죽음의카운트/도발/트집/앵콜/사슬묶기/꿰뚫어보기/미라클아이/저주
- **특이**: 길동무/죽음의카운트 → 분신 파괴 + 빈사 동시; 울부짖기/날려버리기는 차단 불가

### 방어 (+4 우선도)

- 해당 턴 대부분의 기술 차단
- **예외**: 방어관통 기술, 저주/변신/미래예지 등

### 길동무

- 자신 상태; 공격받아 쓰러지면 공격자도 빈사
- 기합의띠 무시; 대타출동 차단 안 됨; 상태이상/날씨 데미지에는 발동 안 됨

### 매직코트

- 변화기술을 매직미러처럼 반사

### 공중날기/구멍파기/다이빙/섀도다이브

- 1T 무적 후 공격; 각각 취약한 기술 존재

## 11. 상태변화 — 상대에게 적용 (Part 2)

### 하품 / 동면

- 다음 턴 종료 시 잠듦/얼음 상태
- 대타출동으로 차단 불가 (최종적인 상태이상은 적용됨)
- 2체 잠듦 한계 시 하품 사용은 되나 잠듦은 적용 안 됨

### 회복봉인

- HP 회복 행위 전부 불가 (기술/아이템/상태이상 회복 모두 포함)
- 멘탈효과

### 사슬묶기

- 마지막 사용 기술 사용 불가
- 멘탈효과 (멘탈허브로 해제 가능)
- 구애 상태에서 마지막 기술이 봉인되면 발버둥만 가능

### 봉인

- 특정 기술만 사용 불가; 교대해도 유지 (멘탈효과 아님)

### 떨어뜨리기

- 지면에 내려앉음; 땅 기술 피격; 압정뿌리기 등 즉시 발동

### 텔레키네시스

- 공중에 뜸; 일격필살 제외 모든 기술 필중
- 교대해도 유지 (마음의눈과 다름)

### 송전 / 악인 / 침수

- 그 턴의 자신 기술이 각각 전기/악/물 타입이 됨

### 미래예지 / 파멸의소원

- 다음 턴 종료 시 데미지 적용
- 사용자의 특공으로 계산; 사용자 능력변화 무시
- 방어 관통; 타입 상성 적용
- 필드에 있는 포켓몬이 KO 크레딧 획득

### 폭탄

- 접촉 기술 받으면 상대에게 1/4 HP 데미지; 턴 종료나 피격 후 자동 만료

## 12. 상태변화 — 자신에게 적용 (Part 2)

### 난동부리기

- 2T 동안 같은 기술 강제 사용
- 정상 만료 시 혼란 발동
- 빗나가거나 풀죽음 등으로 중단 시 혼란 없음
- 마이페이스 특성으로 혼란 방지 가능

### 소란피기

- 난동부리기와 유사하나 만료 시 혼란 없음
- 발동 중 필드 전체 잠듦 방지 (방음 특성은 수면 방지 효과 면역)

### 아쿠아링

- 매 턴 종료 시 최대HP +1/8 (큰뿌리로 강화)
- 회복봉인 상태에서 불가

### 충전 / 냉각 / 가열

- 다음 전기/불꽃/얼음 기술 위력 ×2
- 다른 타입 사용하거나 턴 종료 시 소멸

### 전자부유

- 자신이 공중에 뜸; 지면 타입 무효
- 필중 효과 없음
- 떨어뜨리기, 땅 기술 피격, 교대 시 해제

## 13. 상태변화 — 자신에게 적용 (Part 3)

### 작아지기

- 회피+2 효과는 기술 효과
- 상태 자체는 디메리트: 짓밟기/누르기/하드롤러/드래곤다이브/고스트다이브/플라잉프레스가 필중 + 위력×2

### 웅크리기

- 방+1 효과는 기술 효과
- 상태 자체: 구르기/아이스볼 위력 ×2

### 분노

- 공격기술로 데미지 받을 때마다 공격+1
- 연속기 → 맞은 횟수만큼 상승
- 4~5T 지속 (원작과 다름); 부가 데미지는 발동 안 시킴

### 뿌리박기

- 매 턴 1/16 HP 회복; 강제교대 면역
- 통상교대 불가; 공중에 떠있어도 땅 기술 피격
- 전자부유 사용 불가; 바톤터치/유턴/아름다운허물로는 교대 가능
- 교대 후 다음 포켓몬이 압정 등 영향 받는지는 해당 포켓몬 타입 기준

### 비축하기

- 1~3회 중첩; 방/특방 각 +1씩 상승
- 토해내기 위력: 1회=100, 2회=200, 3회=300
- 꿀꺽 회복량: 1/4, 1/2, 전체
- 토해내기/꿀꺽 사용으로 상태 해제 시 방/특방 상승도 사라짐
- 바톤터치로 교대 시 방/특방 상승은 유지, 상태는 인계 안 됨

### 참기

- 2T 동안 행동 불가
- 상태 해제 턴 종료 시, 받은 데미지×2를 마지막 공격자에게 반환
- 물리/특수 무관 반환

### 날따름

- 상대 필드의 단일 대상 기술을 자신에게 유인
- 아군 기술에는 작용 안 함
- 카운터/미러코트/참기의 반환 대상도 자신으로 변경 가능
- 모으기 기술은 실제 공격 턴에만 유인 가능
- 피뢰침/마중물 특성보다 우선
- 자신이 쓰러지면 원래 대상으로 돌아옴

### 파워트릭

- 공격과 방어 실수치를 교환 (능력변화는 유지)

### 조이기

- 통상교대 불가; 매 턴 종료 시 최대HP 1/8 데미지
- 사용자가 필드를 나가도 유지됨

## 15. 필드의 상태 — 자진 효과

아군 필드에 적용되는 효과들. 중첩 불가 (같은 효과 재사용 불가, 시간만 갱신 안 됨).

### 순풍

- 4T 전개 (실질 효과 3T)
- 아군 속도 ×2
- 안개제거로 해제 불가
- 속도 기반 기술 위력도 ×2

### 리플렉터

- 5T; 상대 물리기술 위력 반감
- 급소 시 효과 상실
- 깨트리다 등에는 효과 없음
- 혼란 자해 데미지는 반감 안 됨
- 사이코쇼크형 기술 (특공 vs 방어) 무효
- 안개제거로 해제 가능

### 빛의장막

- 5T; 상대 특수기술 위력 반감
- 리플렉터와 동일한 특이사항

### 오로라베일

- 싸라기눈 날씨 조건 필요
- 5T; 물리+특수 모두 반감
